# System Requirements
* git >= 2.25.1
* npm >= 8.5.5
* node >= 16.15.0

Made sure to have the correct versions installed.
```shell
git --version
node --version
npm --version
```

# Running the app
1. Using `yarn` install all dependencies.
```bash
  yarn install
```

2. Run the project in locally on development mode.
```bash
  yarn dev
```

# Running the tests
Run all tests.
```bash
  yarn test
```

# Lint and fix files
Check and fix all files.
```bash
  yarn lint
```
